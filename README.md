# Trabalho de Reposição - Git e GitHub  
Autor: Eduardo Seiji Yamamoto da Silva  

## Objetivo
Demonstrar o uso de comandos básicos do Git e GitHub, incluindo commits, branches, merges e o uso do arquivo .gitignore.

## Comandos usados
- `git init`: cria um novo repositório Git local.  
- `git add`: adiciona arquivos à área de stage.  
- `git commit -m "mensagem"`: salva as alterações feitas.  
- `git branch nome-da-branch`: cria uma nova branch.  
- `git checkout nome-da-branch`: muda para a branch indicada.  
- `git merge nome-da-branch`: junta a branch à principal.  
- `git push`: envia tudo para o GitHub.  
- `git pull`: atualiza o repositório local.

## Estrutura do Repositório
- `README.md` → explicação do trabalho.  
- `.gitignore` → lista de arquivos ignorados pelo Git.  
- `/prints` → imagens de comprovação das etapas do trabalho.
